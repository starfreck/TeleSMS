# TeleSMS
This Script is written in Python 3.9 and it built for older Android version and tested on Android 7.0 (Will not work on Android 11 for sure :) Already tested..!!)
Also, it is build only for [Termux](https://termux.com/) and you have to install [Termux:API](https://wiki.termux.com/wiki/Termux:API) and [Termux:Boot](https://wiki.termux.com/wiki/Termux:Boot)


1. Create crond.sh in "boot" which has just crond
2. Add you cron job in "crontab -e"
3. that's it.
